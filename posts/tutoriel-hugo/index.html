Installation du thème m10c gohugo.io

Jan 23, 2021
2 min read
Dev Gohugio Thème-Hugo m10c

Pour mon premier article, j’ai décidé de vous montrer comment installer le thème m10c.

Gohugo c’est quoi ?

“Hugo est un générateur de site statique rapide et moderne écrit en Go et conçu pour rendre la création de site Web à nouveau amusante” traduction de hugo.

Installation

Sous Mac

    brew install hugo

Sous Windows

Si vous utilisez le gestionnaire de paquet Chocolatey

    choco install hugo -confirm

Si vous utilisez le gestionnaire de paquet Scoop

    scoop install hugo

Vérifié si hugo est installé :

    hugo version

Création de notre nouveau Site

    hugo new site [nom_de_notre_site]
    cd [path]
    git clone https://github.com/vaga/hugo-theme-m10c.git themes/m10c

On ajoute le thème dans le fichier config.toml

    theme = "m10c"

Configuration

Dans le fichier config.toml, définir les params:

  • author: Name of the author
  • description: Short description of the author
  • avatar: Path of file containing the author avatar image
  • menu_item_separator: Separator between each menu item. HTML allowed (default: " - “)
  • favicon: Absolute path of your favicon.ico file (default: “/favicon.ico”)

Exemple :

[params]
  author = "Doe John"
  description = "Description website"
  menu_item_separator = " - "

Paramètre Social :

[[params.social]]
  name = "github"
  url = "https://github.com/exemple"

Style :

[params.style]
  darkestColor = "#d35050"
  darkColor = "#212121"
  lightColor = "#f5e3e0"
  lightestColor = "#f5f5f5"
  primaryColor = "#fff"
